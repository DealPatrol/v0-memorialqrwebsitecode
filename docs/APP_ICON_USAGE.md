# Memorial QR App Icon

## Icon Details
- **Size**: 1024x1024 pixels (required by Facebook)
- **Format**: PNG with transparency
- **Location**: `/public/app-icon-1024.jpg`
- **Design**: Elegant memorial theme with QR code element, purple and gold colors

## Where to Use This Icon

### Facebook OAuth Setup
1. Go to your Facebook App settings: https://developers.facebook.com/apps/1207795304536612
2. Scroll to "App icon (1024 x 1024)"
3. Upload the icon from: `public/app-icon-1024.jpg`
4. Click "Save Changes"

### Other Uses
- **Favicon**: Create smaller versions (16x16, 32x32, 192x192) from this master icon
- **PWA Icons**: Use for Progressive Web App manifest
- **Social Media**: Use as profile picture on social platforms
- **App Stores**: If you ever create mobile apps

## Creating Additional Sizes

If you need smaller versions, you can resize this master 1024x1024 icon to:
- 512x512 (standard app icon)
- 192x192 (PWA icon)
- 180x180 (iOS icon)
- 96x96 (Android icon)
- 32x32 (favicon)
- 16x16 (favicon)
